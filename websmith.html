WEBSMITH

Description:You will need to build a website by completing all the subtasks. In total there will be 3 levels. Level 0, Level 1 and bonus level.

Rules:

  • Marking Criteria:
    • User Interface, Responsiveness & Usability - 30 Points
    • Each of the features in level 0 and level 1 - 10 Points (10*7 = 70 Points in total)
    • Features in Bonus Level - 20 Points
    • Security & Bug-free code - 20 Points
    • -Maintaining the git repository - 10 points
  • Submission will be through a google form.
  • You will have to submit the link to the GitHub repository. (Mandatory)
  • It will be a plus point if you deploy it on a hosting platform like Vercel, Heroku, etc., and submit the link of that as well. (Optional)
